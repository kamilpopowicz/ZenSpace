# Contributing

ZenSpace is a personal project, but contributions are welcome.

## How to contribute

1. Fork the repo
2. Create a branch (`git checkout -b feature/your-feature`)
3. Make changes, ensure `swift build` and `swift test` pass
4. Commit with a descriptive message
5. Push and open a Pull Request

## Code style

- Swift 5.10+, SwiftUI
- MVVM architecture
- Use `L("key")` for localized strings (Bundle.module)
- Add tests for new services/ViewModels

## Reporting bugs

Open a GitHub Issue with:
- macOS version
- Steps to reproduce
- Expected vs actual behavior
