# TECH_STACK - ZenSpace

## Core Technologies
- **Language**: Swift 5.10+
- **UI Framework**: SwiftUI (macOS 12.0+)
- **Architecture**: MVVM + Clean Architecture
- **Dependency Management**: Swift Package Manager (SPM)

## System Integrations
- **EventKit**: Calendar & Events integration.
- **MediaPlayer**: Now Playing & Media controls.
- **IOKit / Foundation**: Battery monitoring & System state.
- **XPC**: Inter-process communication for background tasks.
- **AppKit**: Native macOS window management and system sounds.

## UI/UX Components
- **Vortex**: Custom animations and notch integration.
- **GlassEffect**: SwiftUI Blur and material effects.
- **Progressive Blur**: Advanced visual effects for focus modes.

## Security & Storage
- **Keychain**: Secure license key storage.
- **UserDefaults**: Application settings and trial metadata.
- **Notarization**: Required for macOS distribution outside App Store.

## Tools & Utilities
- **SwiftLint**: Code style enforcement.
- **BetterDisplay API**: Third-party monitor control integration.
