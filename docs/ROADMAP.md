# ROADMAP - ZenSpace

## 3.1. Wizja projektu
- **Short-term (1-2 tyg)**: Setup, struktura, podstawowa aplikacja.
- **Mid-term (2-4 tyg)**: Licencje, Kalendarz, Media, Focus Modes.
- **Long-term (4-8 tyg)**: BetterDisplay, Battery, Lock screen sounds, Notarization.

## 3.2. Kamienie milowe (Milestones)
- **M1: Setup**: Struktura projektu, Xcode, Empty UI.
- **M2: Core Data Models**: System modeli i UserDefaults.
- **M3: License System**: Aktywacja, Trial, UI Licencji.
- **M4: Calendar Integration**: Uprawnienia, pobieranie wydarzeń, UI kalendarza.
- **M5: Media/Now Playing**: Kontrola muzyki, metadata, Quick Peek.
- **M6: Focus Modes**: Monitoring systemu, ustawienia trybów.
- **M7: Battery & Lock Screen**: Monitoring baterii, dźwięki systemowe.
- **M8: BetterDisplay Integration**: Detekcja i powiadomienia OSD.
- **M9: UI Polish & UX**: GlassEffect, NotchView, animacje.
- **M10: Settings & Permissions**: Pełne okno ustawień.
- **M11: XPC Helper**: Separacja procesów dla audio/BetterDisplay.
- **M12-M15**: QA, Testy, Notarization, Release.
